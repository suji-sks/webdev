Experiment 1: Hello World & Basics

Aim: To write and run a simple program that prints "Hello, World!".

Objectives

  • Understand the program structure.
  • Learn how to compile and execute a program.
  • Become familiar with print statements and special characters like &, <, >.

Tools Required

Editor
Any text editor (Notepad, VS Code, nano)
Compiler / Interpreter
Python 3.10+ (or any language runtime)

Theory (Related to Experiment)

A simple program demonstrates input/output and program entry point. In many languages, execution starts at a main function or the top-level script. Understanding the flow of execution is fundamental to debugging and program design.

Code (Printout format)

# Python 3 example - Hello World
print("Hello, World!")

# Demonstrating special characters
print("Use & to represent an ampersand, < and > for angle brackets.")

Output (Printout format)

Hello, World!
Use & to represent an ampersand, < and > for angle brackets.

Outcome of experiment

Students will be able to create, save, and run a basic program and observe console output. They will also recognize escaped special characters in text output.

Experiment 2: Loops & Conditionals

Aim: To demonstrate for and while loops and conditional branching.

Objectives

  1. Write loops to iterate over ranges and lists.
  2. Use if / elif / else for decision making.
  3. Combine loops with conditionals to solve simple problems.

Tools Required

  • Text editor
  • Python interpreter

Theory (Related to Experiment)

Loops repeat a block of code. A for loop iterates over a sequence, while a while loop continues until a condition becomes false. Conditionals alter the flow of control based on boolean expressions.

Code (Printout format)

# Python example - loops and conditionals
for i in range(1, 6):
    if i % 2 == 0:
        print(i, "is even")
    else:
        print(i, "is odd")

# Nested loop example
for a in [1, 2]:
    for b in ['x', 'y']:
        print(a, b)

Output (Printout format)

1 is odd
2 is even
3 is odd
4 is even
5 is odd
1 x
1 y
2 x
2 y

Outcome of experiment

Students will be capable of implementing iterative solutions and applying conditionals to control program logic. They will also read and interpret nested loops' outputs.

Experiment 3: File I/O and Exception Handling

Aim: To read from and write to files and handle common I/O exceptions.

Objectives

  • Open, read, and write files safely.
  • Use try/except blocks for graceful error handling.

Tools Required

Editor
Any text editor
Interpreter
Python 3.x

Theory (Related to Experiment)

File I/O allows persistent storage. Always close files or use context managers to avoid resource leaks. Exception handling ensures the program does not crash on runtime errors.

Code (Printout format)

# Python example - File I/O
try:
    with open('sample.txt', 'w') as f:
        f.write('Line 1\nLine 2\n')
    with open('sample.txt', 'r') as f:
        data = f.read()
        print('File contents:')
        print(data)
except IOError as e:
    print('I/O error occurred:', e)

Output (Printout format)

File contents:
Line 1
Line 2

Outcome of experiment

Students will learn to persist data to disk, retrieve it, and handle common errors like missing files or permission issues. This is essential for larger applications.
